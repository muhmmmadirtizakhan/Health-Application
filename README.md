## 📋 Overview 📖
**Health Hub Application** is a modern **healthcare dashboard** that helps users track and manage their health using multiple calculators and analysis tools.  

💡 It provides:
- 📊 Health insights  
- 🧠 Smart calculations  
- 🔐 Secure authentication  

---

## ✨ Key Features 🌟

- 🔐 Secure Login & Registration (Firebase Auth)  
- 📊 Interactive Health Dashboard  
- ⚡ Fast and responsive UI  
- ☁️ Real-time data handling (Firestore)  

---

## 🩺 Health Tools Included 🧠

- ⚖️ BMI Calculator  
- 🔥 BMR Calculator  
- 💪 Body Fat Percentage  
- 🎯 Ideal Weight Calculator  
- ⚡ TDEE Calculator  
- 📏 Waist-Height Ratio  
- 💧 Water Intake Calculator  
- ❤️ Heart Rate Zones  
- 🏋️ Lean Body Mass  
- 📐 Body Surface Area (BSA)
