# 🏥 Health Hub Application  

The **Health Hub Application** is a modern healthcare web platform that helps users track and manage their health with a variety of calculators and monitoring tools.  
It features a **secure login & registration system** powered by **Firebase Authentication**, data management with **Firestore Database**, and is deployed using **Firebase Hosting**.  
The backend is supported by **C++**, while the frontend is built with **HTML, CSS, and JavaScript**, ensuring a clean and responsive user interface.

## 🩺 Health Dashboard Features  

- ⚖️ **BMI Calculator** – assess body mass index and weight category  
- 🔥 **BMR Calculator** – calculate daily calorie needs at rest  
- 💪 **Fat Percentage** – estimate body fat percentage  
- 🎯 **Ideal Weight** – suggested weight range based on height & age  
- ⚡ **TDEE (Total Daily Energy Expenditure)** – estimate daily calories burned  
- 📏 **Waist-Height Ratio** – analyze body fat distribution and risks  
- 💧 **Water Requirement** – recommended daily water intake  
- ❤️ **Heart Rate Zones** – calculate optimal zones for training and cardio  
- 🏋️ **Lean Body Mass** – measure muscle mass vs body fat  
- 📐 **BSA Calculator (Body Surface Area)** – useful in medical and clinical contexts
